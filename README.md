# smart-energy-consumption-monitor

## Problem Description

Energy consumption in households and offices often goes unmonitored, leading to inefficient usage and higher electricity bills.
This project aims to design a system that monitors and analyzes energy consumption of different appliances.
Using efficient data structures, the system can quickly store appliance usage, identify high energy consuming devices, and maintain historical consumption records.

## Data Structures Used

Hash Table
Used to store and quickly retrieve appliance energy consumption data.

Heap (Max Heap)
Used to identify appliances that consume the most energy.

Binary Search Tree (BST)
Used to store and analyze historical energy consumption data.

## Expected Features

Store energy consumption data of appliances
Identify the highest energy consuming devices
Maintain and analyze historical energy usage
Efficient search and retrieval of appliance data
Display energy usage statistics
